⚠ Alcance del módulo
Este módulo diseña la viga de fundación continua sobre base elástica (modelo Winkler).
El análisis estructural se realiza mediante el Método de Elementos Finitos (MEF) con viga Euler-Bernoulli sobre resortes de Winkler: 100 elementos, 101 nodos, rigidez de resorte k = ks·B·Le.

No contempla: empuje lateral del suelo, asentamientos diferenciales entre apoyos, análisis sísmico, ni vigas con secciones variables o discontinuas. El valor ks debe provenir del informe geotécnico o de correlaciones con ensayos de placa.
