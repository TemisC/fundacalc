6.2 — Pilote Individual
Capacidad axial por capas · capacidad lateral · diseño RC sección circular
  • Capacidad axial: método α (arcilla) y β (arena) por capas
  • Punta: Nc·cu (arcilla) o Nq·σ'v (arena)
  • Capacidad lateral: Broms (1964) cabeza libre, cohesivo y granular
  • Diseño RC: barras longitudinales + espiral mínima (ACI 318)

Capacidad axial: método α para arcillas (undrained, API RP2A) y método β para arenas (K·tan δ · σ'v). Capacidad lateral: Broms (1964) con distinción pilote corto (rígido) y largo (flexible) según el momento de fluencia My. El encepado distribuye la carga de la columna entre los pilotes del grupo y diseña la losa de cabeza por punzonado, cortante y flexión.
